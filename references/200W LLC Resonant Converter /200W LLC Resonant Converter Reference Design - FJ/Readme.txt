Known Issues:
None.

Hardware Changes:
To use the updated VMC and ACMC software packages it is required that the following hardware 
changes be made to the LLC hardware (if not already done):
- Short the output inductor L4
- Change R11 - 20 ohm 
- Change R13 - 620 ohm
- Change C11 - 10uF 16V
