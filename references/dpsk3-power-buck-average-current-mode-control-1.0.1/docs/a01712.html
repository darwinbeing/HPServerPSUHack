Detailed Description

System Anti-Windup (Output Clamping) Thresholds.

The NPNZ_LIMITS_t data object holds all parameters required to automatically clamp the most recent control output to user-defined thresholds. This data type allows the definition of individual minimum and maximum output values for the NPNZ controller primary and alternate output port.

This feature is optional and needs to be enabled, configured and managed manually in user code.
