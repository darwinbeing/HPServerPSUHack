# ICSP Header pinout
- 1 - GND
- 2 - MCLR
- 3 - PGED2
- 4 - PGEC2
- 5 - VDD

# Pickit 3 Pinout
- 1 - MCLR
- 2 - VDD
- 3 - GND
- 4 - PGD
- 5 - PGC
